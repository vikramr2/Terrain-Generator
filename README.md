# World Generator
World Generator is a program that generates, a low poly 3D world with grass, water, and a sky!
This experience is made interactive as you can fly around this world
## Controls

| Key      | Action                                                      |
|----------|-------------------------------------------------------------|
| `w`      | Move forward                                                |
| `a`      | Move to the left                                            |
| `s`      | Move to the right                                           |
| `d`      | Move back                                                   |
| `SPACE`  | Ascend                                                      |
| `SHIFT`  | Descend                                                     |
